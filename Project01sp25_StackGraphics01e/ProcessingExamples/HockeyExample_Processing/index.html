References and Resources

Sound Effects

SoundBible.com Sound that are labeled ‘public domain’ or ‘creative commons’ can be used without fee in games. Attribution should be given.
Freesfx is another site with free sound effects.

Sound Editing

Audacity. This is freely downloadable editing software with a high degree of functionality. If saving as MP3 is needed, you’ll have to download the LAME MP3 encoder and install it. Goldwave Freely downloadable sound editor with a large set of audio formats in which sounds can be saved.
LAME MP3 encoding software.

Graphics Editing

Paint. Comes with Windows, and is a highly underestimated tool for putting together 2D images.
LView - An image editor that is a valuable addition to Paint. It is especially useful for making backgrounds in GIF images transparent. Free download, but you should send them money if you like it.
